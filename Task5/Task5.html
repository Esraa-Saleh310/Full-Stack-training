• Here are the very short steps to create an ASP.NET Core project in VS Code:
    1. Install tools Install VS Code + the .NET SDK + the "C# Dev Kit" extension.
    2. Open a folder File → Open Folder → choose an empty folder for your project.
    3. Open the terminal Terminal → New Terminal
    4. Create the project Type one command based on the type you want:
      • MVC app: dotnet new mvc -n MyApp
      • Web API: dotnet new webapi -n MyApp
      • Simple web app: dotnet new web -n MyAp
    5. Go into the project folder cd MyApp
    6. Run it dotnet run Copy the link shown in the terminal (like http://localhost:5000) and open it
    7. Open in browser Open Copy the link shown in the terminal (like http://localhost:5000) and open it.
    8. (Optional) Save to GitHub Open "Source Control" → commit → push to GitHub
